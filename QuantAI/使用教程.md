# QuantAI 团队（Crew ）简介

## Ⅰ.安装

详见：[配置教程.md](https://github.com/Ziyu-Xuuuu/Stock_Quant/blob/main/QuantAI/%E4%BD%BF%E7%94%A8%E6%95%99%E7%A8%8B.md)

### 自定义设置

- 添加你的 `大模型API密钥` 到 `.env` 文件中
- 修改 `QuantAI/src/justatry/config/agents.yaml`来定义你需要的代理
- 修改 `QuantAI/src/justatry/config/tasks.yaml` 来定义你的任务
- 修改 `QuantAI/src/justatry/crew.py`来添加需要的逻辑、工具和特定的参数
- 修改 `QuantAI/src/justatry/main.py`来为代理和任务添加自定义输入

## Ⅱ.运行QuantAI

要启动你的AI代理团队并开始执行任务，请在项目的根文件夹中运行以下命令：

```bash
crewai run
```

## Ⅲ.了解你的团队（Crew）

QuantAI 团队由多个AI代理组成，每个代理都有独特的角色、目标和工具。这些代理在config/tasks中定义的一系列任务上进行协作。利用他们的集体技能来实现复杂的目标。
