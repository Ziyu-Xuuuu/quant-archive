# QuantAI使用教程

本教程将帮助你了解，部署和使用QuantAI这个AI智能体。

## Ⅰ.Introduction

QuantAI 是一个基于人工智能代理的协作框架，它允许多个智能体协同工作，完成复杂任务。具体部署基于CrewAI的基础代码。它的核心理念是将多个相同或不同的大模型代理组织成一个“团队”（Crew），每个代理都有特定的角色、能力和目标，就像一个真正的团队一样协作解决问题。

## Ⅱ.QuantAI 的核心概念

### 1.Agent（代理）

代理是QuantAI的基本单元，每个代理可以有不同的技能、个性和目标。
代理一般基于 LLM（大语言模型）来驱动，比如 OpenAI GPT-4、Claude、Llama 等。

### 2.Task（任务）

每个代理都会被分配任务，任务可以是单独执行，也可以是多个代理协作完成。

### 3.Crew（团队）

一个由多个代理组成的 AI 团队，每个成员可以有不同的职责。
例如，一个市场情绪智能体团队可能包括：
资深量化数据研究员：在数据集中选取重要的评论
量化报告分析师：基于量化数据研究员的研究成果撰写详细报告
QuantAI允许代理与外部工具交互，例如搜索引擎、API、数据库等，使其能力更加强大。

## Ⅲ.搭建QuantAI开发环境

这一部分将帮助你搭建QuantAI的开发环境
打开你为了Stock_Quant的虚拟环境

### 1.安装QuantAI

使用以下任一方法使用所有推荐的工具安装 QuantAI：

```cmd
pip install 'crewai[tools]'
```

&

```cmd
pip install crewai crewai-tools
```

如果其中一种失效可以使用另一种。

若你已经下载过，可以通过以下代码更新:

```cmd
pip install --upgrade crewai crewai-tools
```

### 2.验证安装

检查已安装的版本：

```cmd
pip freeze | grep crewai
```

你将会看到如下内容：

```cmd
crewai==X.X.X
crewai-tools==X.X.X
```

若grep报错，可以直接使用：

```cmd
pip freeze
```

在输出的所有版本中找到：

```cmd
crewai==X.X.X
crewai-tools==X.X.X
```

则证明安装成功！

## Ⅲ.常见问题

### 1.安装CrewAI需要安装 Microsoft C++ 编译工具

下载网站:https://visualstudio.microsoft.com/zh-hans/visual-cpp-build-tools/

所需的组件为:

Microsoft C++ Build Tools:

必须勾选的选项

✅ MSVC v143 - VS 2022 C++ x64/x86 生成工具（必须）

✅ Windows 11 SDK (10.0.22621.0)（建议最新）

✅ 用于 Windows 的 C++ CMake 工具（推荐）

✅ C++ AddressSanitizer（可选，但调试有用）

✅ vcpkg 包管理器（可选）

还需要勾选的额外组件

✅ 适用于最新 v143 生成工具的 C++ ATL（如果你需要 ATL 支持）

✅ 适用于最新 v143 生成工具的 C++ MFC（如果你的项目用到 MFC，建议勾选）

✅ 对 v143 生成工具（最新）的 C++/CLI 支持（如果你需要 C++/CLI 语言支持）

✅ 适用于 Windows 的 C++ Clang 工具（可选）
