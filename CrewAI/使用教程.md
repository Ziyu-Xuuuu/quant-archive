# CrewAI使用教程
本篇文章将帮助你了解，部署和简单的使用CrewAI这个AI工具。

## Introduction
CrewAI 是一个基于 AI 代理（Agent） 的协作框架，它允许多个智能体（AI 代理）协同工作，完成复杂任务。它的核心理念是将 AI 代理组织成一个“团队”（Crew），每个代理都有特定的角色、能力和目标，就像一个真正的团队一样协作解决问题。

## CrewAI 的核心概念
Agent（代理）：

代理是 CrewAI 的基本单元，每个代理可以有不同的技能、个性和目标。
代理可以基于 LLM（大语言模型）来驱动，比如 OpenAI GPT-4、Claude、Llama 等。
Task（任务）：

每个代理都可以被分配任务，任务可以是单独执行，也可以是多个代理协作完成。

Crew（团队）：

由多个代理组成的 AI 团队，每个成员可以有不同的职责。

例如，一个 AI 记者团队可能包括：

研究员（Researcher）——收集数据和信息

编辑（Editor）——整理和优化内容

撰稿人（Writer）——撰写最终的文章

工具集成（Tools & Actions）：

CrewAI 允许代理与外部工具交互，例如搜索引擎、API、数据库等，使其能力更加强大。

## 搭建CrewAI开发环境

这一部分将帮助你搭建CrewAI的开发环境

### 创建虚拟环境
由于CrewAI的使用仍在实验阶段，为不影响主环境，建议使用时先建立一个虚拟环境

Using conda:

```cmd
conda create -n crewai-env python=3.12
```

### 激活虚拟环境

Using conda:

```cmd
conda activate crewai-env
```

### 安装CrewAI

使用以下任一方法使用所有推荐的工具安装 CrewAI：

```cmd
pip install 'crewai[tools]'
```

or

```cmd
pip install crewai crewai-tools
```

如果其中一种失效可以使用另一种。

若你已经下载过，可以通过以下代码更新:

```cmd
pip install --upgrade crewai crewai-tools
```

### 验证安装
检查已安装的版本：

```cmd
pip freeze | grep crewai
```

您应该会看到如下内容：
```
crewai==X.X.X
crewai-tools==X.X.X
```

若grep报错，可以直接使用：

```cmd
pip freeze
```

在输出的所有版本中找到：

```
crewai==X.X.X
crewai-tools==X.X.X
```

安装成功！

## 常见问题
### 安装CrewAI需要安装 Microsoft C++ 编译工具
下载网站:https://visualstudio.microsoft.com/zh-hans/visual-cpp-build-tools/

所需的组件为:

Microsoft C++ Build Tools:

必须勾选的选项

✅ MSVC v143 - VS 2022 C++ x64/x86 生成工具（必须）

✅ Windows 11 SDK (10.0.22621.0)（建议最新）

✅ 用于 Windows 的 C++ CMake 工具（推荐）

✅ C++ AddressSanitizer（可选，但调试有用）

✅ vcpkg 包管理器（可选）

还需要勾选的额外组件

✅ 适用于最新 v143 生成工具的 C++ ATL（如果你需要 ATL 支持）

✅ 适用于最新 v143 生成工具的 C++ MFC（如果你的项目用到 MFC，建议勾选）

✅ 对 v143 生成工具（最新）的 C++/CLI 支持（如果你需要 C++/CLI 语言支持）

✅ 适用于 Windows 的 C++ Clang 工具（可选）
