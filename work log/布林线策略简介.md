## 布林线策略 (Bollinger Bands)

### **基础原理**

布林线通过价格的移动平均值和标准差构建价格波动范围，用于判断市场是否过热（超买）或过冷（超卖）。是类似概率里的正态分布，如果价格高于了中轨线的几个标准差，我们就认为这个价格过高，后续会降低，反之亦然。

### **适用场景**

- **震荡市场**：价格在一定范围内波动时更有效。
- 判断价格波动范围是否过度。

### **特点**

- 提供动态的支持和阻力。
- 容易在趋势市场中误判价格超买或超卖。
- 超参数包络K的取值，中轨线的n取值

### **公式**

* **中轨线 (MA)**：

  $$
  MA = \frac{\sum_{i=1}^n P_i}{n}
  $$
* **上下轨**：
  上轨线：

  $$
  \text{上轨线} = MA + K \cdot \sigma
  $$

  下轨线：

  $$
  \text{下轨线} = MA - K \cdot \sigma
  $$

- \(P_i\)：第 \(i\) 天价格。
- \(K\)：标准差倍数（通常取 2）。
- \(\sigma\)：价格的标准差。

### **交易规则**

1. **卖出信号**：价格突破上轨，超买。
2. **买入信号**：价格跌破下轨，超卖。

### **实现代码**

```python
class BollingerBandsStrategy:
    def __init__(self, window=20, num_std=2):
        """
        初始化布林线策略
        :param window: 滑动窗口大小
        :param num_std: 标准差倍数
        """
        self.window = window
        self.num_std = num_std

    def generate_signals(self, df):
        """
        生成布林线策略交易信号
        :param df: 包含历史数据的 DataFrame，需有 'close' 列
        :return: 包含信号的 DataFrame
        """
        # 计算布林线
        df['ma'] = df['close'].rolling(window=self.window).mean()
        df['std'] = df['close'].rolling(window=self.window).std()
        df['upper_band'] = df['ma'] + self.num_std * df['std']
        df['lower_band'] = df['ma'] - self.num_std * df['std']
  
        # 生成交易信号
        df['signal'] = 0
        df.loc[df['close'] > df['upper_band'], 'signal'] = -1  # 卖出信号
        df.loc[df['close'] < df['lower_band'], 'signal'] = 1   # 买入信号
  
        return df
```
